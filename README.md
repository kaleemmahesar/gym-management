# Gym Management Web App (Frontend Only)

React + Tailwind admin dashboard for a single gym owner (admin-only). Uses dummy seeded JSON data, Redux Toolkit state, and `localStorage` persistence (no backend).

## Admin Login

- **Username**: `admin@gym.com`
- **Password**: `admin123`

## Run locally

```bash
npm install
npm run dev
```

## What’s included

- Dashboard summary cards (members + earnings/expenses/profit)
- Members: add/edit/delete, photo upload (stored as base64), search + Active/Expired filter, auto-expiry by end date, expiring-soon highlight + WhatsApp reminder simulation
- Attendance: daily marking, per-member history, monthly report
- Payments: record payments, payment history, paid/unpaid/partial status within membership period
- Expenses: add/edit/delete with categories + monthly total
- Reports: monthly earnings/expenses/profit + attendance summary

## Persistence

All data is saved under the `localStorage` key `gym_admin_app_v2`.
